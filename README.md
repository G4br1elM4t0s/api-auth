## Instruções

```
  PNPM: o pnpm é um gerenciado de pacote que nem o npm porém ele apresenta alguns difereciais importantes para o desenvolvimento usando a plataforma nodejs... exemplo alta velocidade na instalação de pacotes, melhor gerenciamento de pacotes e desacoplamento no momento da instalação tornando os pacotes mais leves.
```

```
  Para poder instalar o pnpm basta digitar na CLI do seu terminal o comando npm install -g pnpm
```
